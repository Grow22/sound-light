# 필요한 그림/표 목록

## 그림

### 그림 1. 시스템 전체 구조도
- **내용**: 2단계 분류 파이프라인 흐름도
- **구성**:
  ```
  마이크 입력 → [1단계: YAMNet 521개] → 직접 매핑 (5개)
                                      → 2차 분류기 (5개)
                                      → 무시 (나머지)
  → 서버 전송 → LED/진동 알림
  ```
- **기존 자료**: docs/system_architecture.png (있지만 2단계 구조 반영 필요)
- **상태**: 수정 필요 (현재 구조에 맞게 업데이트)

### 그림 2. Hub-Node 하드웨어 구조 (선택)
- **내용**: RPi Hub + ESP32 Node 배치도
- **구성**: 각 방의 ESP32 → WiFi → 중앙 RPi → 알림 분배
- **상태**: 논문 분량에 따라 포함 여부 결정
- 3페이지에서는 생략하고 그림1에 간략히 표현하는 것이 나을 수 있음

## 표

### 표 1. 카테고리별 분류 정확도
- **내용**: 11개 카테고리 Precision / Recall / F1-Score
- **기존 자료**: docs/confusion_matrix.png에서 수치 추출
- **형식**:
  ```
  카테고리    | Precision | Recall | F1-Score
  ----------|-----------|--------|--------
  가전제품   |    ...    |  ...   |   ...
  개 짖음    |    ...    |  ...   |   ...
  ...
  전체 평균   |    ...    |  ...   |  90.5%
  ```
- **상태**: confusion_matrix.png에서 수치 확인 필요

### 표 2. 2단계 분류 구조 카테고리 매핑
- **내용**: 직접 매핑 5개 + 2차 분류기 5개 정리
- **형식**:
  ```
  분류 방식     | 카테고리           | YAMNet 클래스 예시
  ------------|-------------------|------------------
  직접 매핑(5개) | 개 짖음            | Dog, Bark
              | 물 소리            | Water
              | 인터폰             | Doorbell, Ding-dong
              | 노크 소리           | Knock, Tap
              | 창문 깨지는 소리     | Glass, Shatter
  2차 분류기(5개)| 도어락             | Door, Slam
              | 인터폰             | Telephone, Buzzer
              | 가전제품            | Microwave, Fan
              | 사이렌             | Alarm, Siren
              | 아기 울음           | Crying, Baby cry
  ```
- **상태**: 작성 가능

### 표 3. 기존 솔루션 비교 (선택)
- **내용**: Google Sound Notifications / Earzz / 소리우산 / Hearo 비교
- **형식**:
  ```
  솔루션              | AI 분류 | 다중 공간 | 한국 가정음 | 알림 방식      | 비용
  -------------------|--------|---------|----------|-------------|------
  Google Sound Notif. | O      | X       | X        | 스마트폰만    | 무료
  Earzz (UK)         | O      | O       | X        | 앱 알림      | ~€499
  소리우산 (KR)       | X      | X       | -        | 볼륨 기반    | 유료
  Hearo (제안)       | O      | O       | O        | LED+진동+푸시 | ~25만원
  ```
- **상태**: 작성 가능, 분량에 따라 포함 여부 결정
